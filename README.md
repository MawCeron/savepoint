# Savepoint

An open-source desktop **Attention Interrupt System** for people who
experience deep hyperfocus. Savepoint creates deliberate, hard-to-ignore
interruptions at scheduled moments, gives you a recovery window to regain
conscious attention, then lets you continue.

## Status

Pre-release, v0.1 in progress. See the [roadmap](docs/PROJECT_SCAFFOLD.md#roadmap).

## Tech stack

Avalonia UI (code-behind, no MVVM) on .NET Native AOT, with SQLite via plain
ADO.NET. Targets Linux and Windows initially, macOS planned. See
[docs/ADR.md](docs/ADR.md) for the reasoning behind each choice.
